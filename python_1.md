
# 파이썬 시작하기

- Python 스크립트 실행

- 코드 스타일 가이드('," 일치시키기, 할당할 때 띄어쓰기, tab 맞추기)

- **주의사항**

    - 대소문자 구분

    - 띄어쓰기, 문장부호

    - 들여쓰기 : 문장 구분할 때, space 키 4번 or tab 키 1번

- 주석 (Comment)

    - 코드에 대한 설명

    - 앞에 # 나 단축키 ctrl + /

## 필수 단축키

- 파일 만들기 : ctrl + n (new)

- 파일 저장 : ctrl + s (save)

- 주석 처리하기 : ctrl + /
